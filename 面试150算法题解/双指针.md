# 问题1：验证回文串(简单)

## 思考过程
**1、删除非字母数字字符，并转为小写：**
利用ASCII码判断

**2、比较正反是否一致：**
设置前后双指针，逐位判断（终止条件：i < j）

## 参考题解
**1、思路更优：**
直接使用双指针，若为字母数字则判断

**2、方法更优：**
isalnum(s[i])，判断是否为字母数字；
tolower(s[i++])，将大写字母变为小写；

## 代码
### 第一版：
```C++
class Solution {
public:
	bool isPalindrome(string s) {
		vector<char> a;
		int n = 0;
		for (int i = 0; i < s.size(); i ++)
		{
			bool isZS = ((s[i] >= 48 && s[i] <= 57) || (s[i] >= 65 && s[i] <= 90) || (s[i] >= 97 && s[i] <= 122));
			if (isZS) {
				if (s[i] >= 65 && s[i] <= 90) {
					a.push_back(s[i]+32);
				}
				else {
					a.push_back(s[i]);
				}
			}
		}
		int i = 0, j = a.size();
		while (i < j) {
			if (a[i] != a[j])
				return false;
			i++; j--;
		}
		return true;
	}
};
```

### 第二版：
```C++
class Solution {
public:
    bool isPalindrome(string s) {
        int i = 0;
        int j = s.size() - 1;
        while(i < j){
            if(!isalnum(s[i]))
                i++;
            else if(!isalnum(s[j]))
                j--;
            else if(tolower(s[i++]) == tolower(s[j--]))
                continue;
            else
                return false;
        }
		return true;

    }
};
```

## 总结：
**1、ASCII码范围：**

48-57：0-9
65-90：A-Z
97-122：a-z

**2、string内置函数(Python也有这两个名字不同)：**

isalnum()：判断是否为字母数字

tolower()：大写变小写

## 代码
### 第一版：
```C++
class Solution {
public:
	bool isSubsequence(string s, string t) {
		int i = 0;
		int j = 0;
		while (i < s.size()) {
			if (j == t.size())
				return false;
			if (s[i] == t[j]) {
				i++; 
				j++;
			}
			else
				j++;
		}
		return true;
	}
};
```

### 第二版：
```C++
class Solution {
public:
	bool isSubsequence(string s, string t) {
		int i = 0;
		int j = 0;
		while (i < s.size() && j < t.size()) {
			if (s[i] == t[j]) {
				i++; 
				j++;
			}
			else
				j++;
		}

	}
```

# 问题3：两数之和 II - 输入有序数组 (中等)

## 思考过程
**1、直接暴力遍历，超时（不在赘述）**

## 参考题解
**思维：把查找与哪个数相加的和 = target， 改成查找求和后 = target 的数，即可利用优化的查找算法**

## 代码
### 第一版：
暴力查询，超时，不再赘述。

### 第二版：
```C++
class Solution {
public:
    vector<int> twoSum(vector<int>& numbers, int target) {
        for(int i = 0; i < numbers.size(); i++){
            int low = i + 1;
            int high = numbers.size() - 1;
            int x = target - numbers[i];
            while(low <= high){
                int mid = (high - low)/2 + low;
                if(numbers[mid] == x)
                    return {i+1, mid + 1};
                else if(numbers[mid] > x){
                    high = mid - 1;
                }else{
                    low = mid + 1;
                }
            }  
        }
        return {-1, -1};
    }
};
```

## 总结：
**学会换角度思考，不要急于暴力求解，多利用题干提供的信息（递增数列），对此就要先想到查找算法**


# 问题4：盛最多水的容器 (中等)

## 思考过程
**1、还是想的暴力，更优方法没想到，一直想着正向求解，从小逐渐寻找大的**

## 参考题解
**1、找到一个方法，能依次排除掉非最优选项：**
通过找到一个缩小范围的方法并且一定不会漏掉其他答案
正如题解：从边界a,b开始，选择 min(a,b) 往前移，因为无论下一个边界大小，一定不会比 max(a,b)往前移 的结果大
如果移动大端，未来的高度，一定 >= min(a,b)，并且宽度-1
所以我们就找到了在一定情况下的最优，并且能依次排除掉非最优选项

## 代码
```C++
class Solution {
public:
    int maxArea(vector<int>& height) {
        int low = 0;
        int high = height.size() - 1;
        int maxSize = 0;
        while(low < high){
            maxSize = max(maxSize, (high-low) * min(height[low], height[high]));
            if(height[low] <= height[high])
                low++;
            else
                high--;
        }
        return maxSize;
    }
};
```

## 思维：
**1、避免重复判断**
